## Summary

The Balboa Gateway Ultra case study applies the IoT Integrator agent
(`docs/prompts/iot-integrator-prompt.md`) to the **Balboa ControlMySpa
Gateway Ultra Wi-Fi module (BWG model 59303)**. It is the second run of
the self-augmenting IoT Integrator stage and the heavier of the two
new IoT-integrator cases.

- **Outcome shape:** configuration-only Phase 3 wrapping the upstream
  MQTT bridge `[REDACTED:repo-path:BALBOA-UPSTREAM-1]`, plus a six-control
  hardening overlay (C-1..C-6).
- **Authentication shape:** AWS Cognito us-west-2 with default 1-hour
  access / 30-day refresh tokens; cryptographically sound but the
  operational layer (broken intermediate-CA chain since June 2023,
  `TrustAllStrategy` symbol on the classpath, public mobile client
  secret, no in-app revocation UI) is not.
- **Obscurity-vs-authentication position:** *operational-obscurity
  exemplar* — sits between Spider Farmer (no auth) and Ondilo (clean
  OAuth2). Two new technique tags proposed for the next-run inventory:
  `T-CROSS-VENDOR-CORPORATE-FLOW` (BWG ↔ WaterGuru inside Helios) and
  `T-OPERATIONAL-OBSCURITY`.
